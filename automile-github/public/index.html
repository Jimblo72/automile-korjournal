Exportera från Automile
1
Öppna Automile

Logga in på app.automile.com → gå till Resor.

2
Välj datumintervall

Filtrera på önskat intervall — t.ex. ett kvartal eller hela året.

3
Klicka Export

Kuvert-/nedladdningsikonen uppe till höger. Välj CSV. Ladda ner filen.

4
Importera här

Dra och släpp Excel-filen (.xlsx) i rutan till höger eller klicka för att välja.

Importera fil
📄

Klicka eller dra hit
Excel eller CSV från Automile

Stöder Automiles Excel-export (.xlsx) och CSV. Befintliga körningar ersätts vid ny import av samma period.

Lägg till manuellt
Kända adresser

Tre typer: Kontor/Tjänst, Hemadress och Privat övrigt.

Klassificeringslogik
✅ Auto → Tjänst
Kontor → var som helst (utom Hem). Eller: var som helst → Kontor (utom från Hem).
🏠 Auto → Privat
Hem↔Kontor (pendling), Hem→okänd, Hem→Hem och kända privatadresser.
❓ Oklassad
Ingen adress matchar. Klicka på badge för att byta typ manuellt.
Kör Omklassificera i körningsvyn efter att du lagt till nya adresser.
